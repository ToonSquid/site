ToonSquid 2.0.4

21 February 2025


By popular demand, the ToonSquid 2.0.4 update reintroduces the vector layer from ToonSquid 1 as the default drawing destination for vector brushes instead of always creating vector groups. It also disables the automatic vector group creation when adding path layers via the path tool or the shape library by default.

Vector groups continue to be a powerful feature in ToonSquid 2.0 and you can choose to opt in to only working with vector groups in the ToonSquid settings by changing the General > Vector Workflow Preference setting to Vector Group and by enabling the Tools > Path > Create Vector Groups setting.

You can now also choose to manually convert any vector layer to a vector group in order to access, edit and potentially animate the individual shapes inside, and you can convert vector groups to vector layers if you prefer to work with a singular layer instead of a group. Tap the vector layer / vector group and choose Create Vector Group / Create Vector Layer.

If you had previously opened an older project from ToonSquid 1 in ToonSquid 2 which had converted all of your vector layers to vector groups, you can convert all of them back into vector layers at once by following these steps:

  1. Open the project.
  2. Tap on Actions at the top.
  3. Tap on Settings.
  4. Select the Info tab on the left.
  5. Scroll down and select Convert Vector Groups to Vector Layers.
