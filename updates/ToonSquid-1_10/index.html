ToonSquid 1.10

29 September 2023


ToonSquid 1.10 is out now and it comes with many exciting improvements. Here are the most important ones:

Crop

  • Change the canvas size after a project or animation clip has been created via Settings → Info → Crop.

Folders

  • Organize your projects and assets using folders in the library.

Transform Tool

  • Edit multiple selected drawings at once using the transform tool.
  • Out of Pegs: Also known as "Shift and Trace", this feature lets you temporarily move your onion skins to help you draw inbetweens while keeping consistent proportions.

Brushes

  • New pencil brushes: "Sketch Pencil" and "Sandy"
  • New brush settings: Tilt Size Compression, Tilt Control Stretch, Tilt Control Shift, Gradation Tilt Control, Bleed Tilt Control
  • The tilt threshold is now a property of each brush and not a global setting anymore.
  • Updated the following brushes: "Pencil", "6B Pencil", "Flat Brush", "Blotchy", "Dry Watercolor". You need to manually reset those brushes if you want the updates to be applied to them.

Selection

  • Polygon Selection: The freehand selection tool mode now allows you to tap individual points to create polygonal selections and doesn't close the selection region automatically anymore. The old behaviour can be restored via Settings → Tools → Selection → Manually close selections.

Gestures

  • Use the Select multiple action as an alternative to the double-tap-and-drag gesture to select multiple drawings or keyframes in the timeline. The gesture can now also be disabled in the gesture settings.
  • You can now disable the quick pinch gesture for resetting the canvas zoom.
  • The pinch-to-zoom gesture in the timeline now feels a lot more reliable.

Other

  • Reverse Drawings: Multiple selected drawings in the same layer can now be reversed using the "Reverse Drawings" action.
  • Canvas Flip: Temporarily flip the canvas horizontally and vertically (Settings → General).
  • ProRes 4444: When exporting an MOV video with transparency included, you can now optionally choose the ProRes 4444 codec, which has better compatibility with some third-party software than HEVC.
  • Camera Outside Darkening: The outside regions of the camera layer are now darkened during playback to make it easier to see what will be visible in the final export and what won't. This can also be enabled during editing and the darkening amount is configurable via the inspector.
  • The font "Weight" and "Style" dropdowns have been combined into a single dropdown that now contains all font variants of an installed typeface.
  • Left-handed UI mode
  • Various bug fixes
  • The minimum supported iPadOS version is now 15.0. (All iPad models running iPadOS 13 and 14 can be updated to iPadOS 15)
