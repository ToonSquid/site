ToonSquid 1.8

08 March 2023


ToonSquid 1.8 is out now and it comes with many exciting improvements. Here are the most important ones:

Fill

  • Dynamic fill threshold adjustment: Touch the screen and drag your finger left and right to change the fill threshold and get a live-preview of the result
  • Gap closing fill (for pixel layers)
  • Added an option to sample from all layers in the fill tool

Brushes

  • Added an Auto Fill brush setting
  • Added a Draw Behind brush setting
  • Added optional brush cursor outlines

Other

  • You can now clear the selected region on pixel and vector layers
  • Separate Copy, Cut and Paste buttons for selections
  • Added an option to automatically merge pixel and vector layers after pasting
  • It is now possible to transform, copy and cut selection regions in vector layers
  • Vector layers remain vector layers after merging
  • Animation layers can now be merged
  • Import multiple files at once into the library
  • Made the backup reminder less interruptive
  • Added alternate editing options during vector editing when touching the screen with a second finger
  • The easing curves of multiple selected keyframes can now be edited at once
  • Added a slider to adjust the selection mask visibility
  • Bug fixes
